# AGENTS

このファイルは本リポジトリにおける開発ルールを定義する。

## 必須ルール

1. すべての関数に対して、`JSDoc` または `TSDoc` を**日本語**で必ず記載する。
2. コミットメッセージは、prefix（例: `feat:`, `fix:`, `chore:` など）を除き、本文を**日本語**で記載する。
3. `ユニットテスト` と `結合テスト` は必須とし、機能追加・修正時は両方を整備する。
4. 実装時は常に `結合度` と `凝集度` を意識し、責務に応じて適切にモジュール化する。
5. `README.md` と `AGENTS.md` は常に最新化する。内容が古い・不正確な場合は、明示的な指示がなくても更新する。

## 運用方針

- 上記ルールに反する変更はマージしない。
- 仕様変更時は実装変更と同時にドキュメント（`README.md`, `AGENTS.md`）を更新する。
- テストが不足している場合は、実装より先に不足観点を洗い出し、追加する。
